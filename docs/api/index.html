ErrorHandler

  • ErrorHandler(...errors: Function[]): ErrorHandlerMethodDecorator
    • Defined in errors/ErrorHandlerDecorator.ts:39

    Errorhandler decorator; decorates the given function as an error handler for the current controller. You can specify various errors in the open parameter list. The handler is registered for the given error types, and is only called if the thrown error matches the registered type. If errors are omitted, the handler will be registered as 'default' and is called if no other specialized handler matches the thrown error.

    Parameters

    • Rest ...errors: Function[]

      List of error classes to register to.

    Returns ErrorHandlerMethodDecorator
